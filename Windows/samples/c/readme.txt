##编译指南：
# CMAKE 快速使用
默认使用TDM-GCC64编译
在CMakeList.txt 所在目录建立文件夹build， 编译相应工程。
1$ cd build
2$ cmake .. -G "MinGW Makefiles"     //Windows 平台
3$ make -j #or mingw32-make according to your environment

####################################################################
注意：
1.可以手动修改CMakeList.txt中的sample选择编译不同的测试样例。
2.将对应版本的库拷贝到可执行文件目录
3.第一次运行会自动加载校准数据，等待板子上红灯闪烁结束后进行后续操作。(1.0版本)
####################################################################
