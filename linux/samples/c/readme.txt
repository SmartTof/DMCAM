##编译测试sample方法

1.进入当前目录，mkdir build,进入build目录
2.cmake .. -G "Unix Makefiles"
3.make -j
####################################################################
注意：
1.可以手动修改CMakeList.txt中的TEST_SRC选择编译不同的测试样例。
2.第一次运行会自动加载校准数据，等待板子上红灯闪烁结束后进行后续操作。
####################################################################
